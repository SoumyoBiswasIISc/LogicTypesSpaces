Sets and Types

A set is a collection of objects with no further structure. Thus, the only property (or function) associated with a set $S$ is membership - a different object $x$ may or may not belong to $S$ (denoted as usual $x\in S$).

It is natural to try to reverse this and consider, for any property of objects (for example sets) the set that consists of those satisfying this (for example all sets that have a certain property).
